# Collision-Detection-System

A robust and efficient **collision detection system** for 2D and 3D environments, designed to detect and handle object interactions in real-time applications and games.

---

## Features

- ✅ Real-time collision detection for 2D and 3D objects  
- ✅ Supports multiple shapes: AABB, Circle, Polygon, Mesh  
- ✅ Efficient algorithms for minimal performance overhead  
- ✅ Customizable collision response handling  
- ✅ Easy integration into game engines and simulations
